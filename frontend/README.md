# Little Garden - Frontend

Frontend game client for Little Garden, built with Phaser.js and Vite.

## Installation

1. Install dependencies:
```bash
npm install
```

## Development

Run the development server:
```bash
npm run dev
```

The game will be available at `http://localhost:5173` (or the next available port if 5173 is in use - check the terminal output for the actual URL)

## Build

Build for production:
```bash
npm run build
```

The built files will be in the `dist` directory.

## Preview Production Build

Preview the production build:
```bash
npm run preview
```
